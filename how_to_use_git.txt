コマンドプロンプト
cd フォルダ名
フォルダ名を指定する
↓
code ファイル名
ファイルを作成する
↓
git init
gitを初期化するコマンド
※gitが作成されてしまうので今後使わない
↓
(git status)
コミットされているか確認
↓
git add　.(手元にあるすべてのファイル)
ステージングに一旦置いて、ローカルレポジトリに置くか選択する
※どこの階層にいるか確認
↓
git commit -m '*コメント*'
ローカルレポジトリにコミット出来た
↓
git push origin main
ローカルの変更点をリモートへ反映する
